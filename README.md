1. Zip this entire folder (everything inside web-avatar).
2. Go to [app.netlify.com](https://app.netlify.com/drop)
3. Drag the zip onto the page.
4. Wait for deployment and copy the live link (e.g. [yourname.netlify.app](https://yourname.netlify.app)
5. Open on your phone and grant camera access.
6. Tap browser menu → 'Add to Home Screen' for full-screen use.
